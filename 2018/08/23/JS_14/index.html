  • 대괄호 안에는 접근하려는 프로퍼티의 속성을 문자열 형태로 적어야 한다.
    하지만 자바스크립트 엔진이 [] 연산자 내에 숫자가 사용될 경우,
    해당 숫자를 자동으로 문자열 형태로 바꿔준다.
